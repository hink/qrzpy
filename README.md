qrzpy
=====

Simple QRZ Script to lookup callsigns via the CLI.

## Requirements

### Python Modules
+ BeautifulSoup 4  
+ Requests  

		pip install beautifulsoup4 requests

### QRZ Requirements
Paid membership with access to the [XML Logbook Data API](https://ssl.qrz.com/products/index.html)

## Python 2 Usage
	python ./qrz.py

## Python 3 Usage
	python ./qrz3.py
